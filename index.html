I am also actively working on LeanTeX, a LaTeX package for embedding Lean 4 code directly in LaTeX and rendering Lean's infoview output in the compiled PDF. The source code is here: GitHub.

GraphToTeX


GraphToTeX is a small tool for converting graphs into LaTeX/TikZ code. It allows users to quickly draw graphs in the browser and export clean TikZ code for use in mathematical documents and textbooks.
